# MiniStore

MiniStore — учебный проект интернет-магазина, сверстанный по Figma-макету на чистом HTML, CSS и JavaScript.

## Демо

[Посмотреть сайт](https://ministore-five-alpha.vercel.app/)

## Макет

[MiniStore — Mini Ecommerce Store Website Template](https://www.figma.com/design/d4x6l6QlFZvySghF9z50ru/MiniStore---Mini-Ecommerce-Store-Website-Template--Community-?node-id=17-72&t=7AVgMURSFuVTw7wZ-0)

## Технологии

- HTML
- CSS
- JavaScript
- Git / GitHub
- Vercel

## Функционал

- Главная страница
- Страница каталога
- Страница “О нас”
- Рендер товаров через JavaScript
- Фильтрация по категориям и тегам
- Поиск товаров
- Добавление товаров в корзину
- Счётчик товаров в корзине
- Корзина реализована через замыкание
- Интерактивные элементы интерфейса

## Что практиковал

- Верстку по Figma-макету
- Семантическую HTML-структуру
- CSS и BEM
- Работу с DOM
- Обработку событий
- Методы массивов: `map`, `filter`, `find`, `forEach`, `includes`, `slice`
- `querySelector`, `closest`, `dataset`
- Использование `Set`
- Разделение JavaScript-кода на модули
- Деплой проекта на Vercel

## Цель проекта

Показать базовые навыки работы с HTML, CSS и JavaScript без использования фреймворков.
